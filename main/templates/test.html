  • STATIC_URL - URL для использования при обращении к статическим файлам, расположенным в STATIC_ROOT
  • STATIC_ROOT - Абсолютный путь к каталогу, где collectstatic будет собирать статические файлы для развертывания.
  • STATIC_DIRS - Этот параметр определяет дополнительные местоположения, которые приложение staticfiles будет проходить, если включен искатель FileSystemFinder , например, если вы используете команду управления collectstatic или findstatic или используете представление обслуживания статических файлов.
  • STATIC_STORAGE - Механизм хранения файлов, используемый при сборе статических файлов с помощью команды collectstatic management.
  • STATIC_FINDERS - По умолчанию будут найдены файлы, хранящиеся в параметре STATICFILES_DIRS (с использованием django.contrib.staticfiles.finders.FileSystemFinder ) и в static подкаталоге каждого приложения (с использованием django.contrib.staticfiles.finders.AppDirectoriesFinder ). Если присутствует несколько файлов с одинаковым именем, будет использован первый найденный файл.
